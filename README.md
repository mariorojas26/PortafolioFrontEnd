# PortafolioFrontEnd

Experiencia web interactiva para el portafolio de Mario Rojas.

## Comandos

```bash
npm run dev
npm run build
npm run preview
```

## Estructura actual

- `src/components/RoomPortfolioHero.tsx`: escena principal del escritorio interactivo.
- `public/assets/Habitacion`: assets visuales usados por la experiencia.
- `src/lib/gsap.ts`: registro central de GSAP.
